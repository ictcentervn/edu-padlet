# 바이브코딩 프롬프트 모음 — Step 5 (Codespaces)

> **대상:** 중학교 1학년 ~ 고등학교 3학년
> **사용 환경:** GitHub Codespaces (브라우저 VS Code)
> **AI 도구:** Claude.ai, ChatGPT, 또는 Codespace 내 Copilot Chat

---

## 이 단계에서 바이브코딩으로 할 수 있는 것

Codespaces에서는 AI에게 물어보고 → 코드를 수정하고 → 바로 결과를 확인할 수 있습니다.

```
  AI에게 물어보기 → 코드 복사 → Codespace에 붙여넣기 → 저장 → 새로고침 → 확인!
```

localhost에서는 나만 보았지만, Codespaces에서는 **친구에게 바로 보여줄 수 있어요.**

---

## 🟢 기본 프롬프트 — 디자인 바꾸기

### 프롬프트 1: 배경색 변경

**Claude.ai 또는 ChatGPT에 아래를 붙여넣으세요:**

```
내 웹 앱의 배경색을 바꾸고 싶어요.
지금 public/style.css 파일에서 background-color가 #f0f0f0인데,
부드러운 파스텔 톤으로 바꿔주세요.
3가지 색상 옵션을 CSS 코드로 알려주세요.
```

**적용 방법:**
1. AI가 알려준 CSS 코드를 복사
2. Codespace에서 `public/style.css` 열기
3. background-color 부분을 수정
4. `Ctrl + S`로 저장
5. 앱 탭에서 새로고침 → 바로 확인!

---

### 프롬프트 2: 카드 모양 꾸미기

```
내 패들렛 앱의 카드 디자인을 더 예쁘게 바꿔주세요.
지금은 평범한 사각형인데, 아래 효과를 추가하고 싶어요:
- 둥근 모서리 (border-radius)
- 마우스를 올리면 살짝 떠오르는 효과 (hover shadow)
- 부드러운 그림자 (box-shadow)
CSS 코드만 알려주세요.
```

---

### 프롬프트 3: 글꼴 변경

```
내 웹 앱의 글꼴을 구글 폰트로 바꾸고 싶어요.
한국어가 예쁘게 보이는 글꼴로 추천해주세요.
HTML의 head에 넣을 link 태그와 CSS의 font-family 코드를 알려주세요.
```

## 🟡 중급 프롬프트 — 기능 추가

### 프롬프트 4: 이모지 반응 기능

```
내 패들렛 앱의 카드에 이모지 반응 기능을 추가해주세요.

요구사항:
- 각 카드 아래에 👍 ❤️ 😂 3개 이모지 버튼이 있음
- 이모지를 클릭하면 숫자가 1 올라감
- Supabase cards 테이블에 reactions 컬럼 추가 (JSON 형식)
- 서버에 PATCH /api/cards/:id/react 엔드포인트 추가

현재 기술 스택:
- Express.js 서버 (server.js)
- Supabase 데이터베이스 (db.js)
- 바닐라 HTML/CSS/JS 프론트엔드 (public/ 폴더)
```

**적용 방법:**
1. AI가 알려주는 코드를 파일별로 복사
2. Codespace에서 해당 파일을 열고 수정
3. 서버 재시작: 터미널에서 `Ctrl + C` → `node server.js`
4. 앱 탭에서 새로고침 → 이모지 버튼 확인!

---

### 프롬프트 5: 다크 모드 토글

```
내 패들렛 앱에 다크 모드 토글 버튼을 추가해주세요.

요구사항:
- 화면 오른쪽 상단에 🌙/☀️ 버튼
- 클릭하면 다크 모드 ↔ 라이트 모드 전환
- 다크 모드: 어두운 배경(#1a1a2e), 밝은 글자(#eee)
- 라이트 모드: 원래 디자인 유지
- CSS 변수(custom properties) 사용
- localStorage로 선택 저장 (새로고침해도 유지)

public/index.html, public/style.css, public/app.js를 수정해주세요.
```

---

### 프롬프트 6: 카드 검색 기능

```
내 패들렛 앱에 카드 검색 기능을 추가해주세요.

요구사항:
- 보드 상단에 검색창 추가
- 제목 또는 내용에 검색어가 포함된 카드만 보여줌
- 클라이언트 측 필터링 (JavaScript)
- 검색어 입력할 때마다 실시간 필터링 (keyup 이벤트)
- 검색어를 지우면 모든 카드가 다시 보임

public/index.html과 public/app.js를 수정해주세요.
```

## 🔴 심화 프롬프트 — 직접 도전

### 프롬프트 7: 카드 드래그 앤 드롭

```
내 패들렛 앱에 카드 드래그 앤 드롭 기능을 추가해주세요.

요구사항:
- 카드를 마우스로 잡아서 다른 위치로 이동 가능
- HTML5 Drag and Drop API 사용 (외부 라이브러리 없이)
- 이동 후 순서가 서버에 저장됨
- PATCH /api/cards/:id/position 엔드포인트 추가
- Supabase cards 테이블에 position 컬럼 추가 (integer)
- 모바일에서도 동작하면 좋겠지만 필수는 아님

현재 기술 스택: Express.js + Supabase + 바닐라 JS
```

---

### 프롬프트 8: 보드 여러 개 만들기

```
내 패들렛 앱을 확장해서 보드를 여러 개 만들 수 있게 해주세요.

요구사항:
- 메인 페이지에서 보드 목록이 보임
- "새 보드" 버튼으로 보드 생성 (보드 이름 입력)
- 보드를 클릭하면 해당 보드의 카드만 보임
- Supabase에 boards 테이블 추가 (id, name, created_at)
- cards 테이블에 board_id 컬럼 추가
- 서버에 /api/boards 엔드포인트 추가

현재 기술 스택: Express.js + Supabase + 바닐라 JS
파일 구조: server.js, db.js, public/index.html, public/style.css, public/app.js
```

## Git 관련 프롬프트

Codespace에서 코드를 수정한 후 GitHub에 올리는 방법도 AI에게 물어볼 수 있습니다.

### 프롬프트 9: 수정한 코드 GitHub에 올리기

```
GitHub Codespace에서 코드를 수정했어요.
이 변경사항을 GitHub 저장소에 올리는 방법을 알려주세요.
터미널 명령어를 단계별로 알려주세요.
git add, commit, push 순서로요.
```

**정답 미리보기:**
```
git add .
git commit -m "기능 추가: 카드 디자인 개선"
git push
```

## AI가 만든 코드 이해하기

AI가 코드를 만들어줬다면, 이해하기 위해 다시 물어보세요:

### 이해 프롬프트

```
이 코드에서 아래 부분이 하는 일을 중학생이 이해할 수 있게 설명해줘:
[코드 붙여넣기]
```

### 에러 해결 프롬프트

```
Codespace 터미널에서 이 에러가 나왔어요. 원인과 해결법을 알려주세요:
[에러 메시지 붙여넣기]
```

## 바이브코딩 팁

1. **구체적으로 요청하세요.** "예쁘게 만들어줘" 대신 "둥근 모서리, 그림자 효과 추가해줘"
2. **현재 기술 스택을 알려주세요.** "Express.js + Supabase + 바닐라 JS"
3. **파일 이름을 알려주세요.** "public/style.css를 수정해줘"
4. **한 번에 하나씩 요청하세요.** 여러 기능을 한꺼번에 요청하면 엉킬 수 있어요.
5. **에러가 나면 에러 메시지를 통째로 복사해서 AI에게 보여주세요.**
