> **선수 학습:** Step 4 (GitHub) 완료 필수

## 오늘 배울 내용

> 브라우저 하나만 있으면 GitHub에 있는 코드를 바로 열어서 실행하고, 링크로 친구에게 공유할 수 있습니다. Node.js 설치도, VS Code 설치도 필요 없습니다.

## 생각해보기

학교 컴퓨터에 Node.js를 설치할 수 없다면 어떻게 할까요?

- 컴퓨터실에서 "프로그램 설치 금지"라고 되어 있다면?
- 집에 컴퓨터가 없고 태블릿만 있다면?
- 친구에게 "내 앱 볼래?" 했는데, localhost:3000은 내 컴퓨터에서만 된다면?

Step 4에서 우리는 코드를 GitHub에 올렸습니다.
그런데 그 코드를 **실행**하려면 여전히 내 컴퓨터에 Node.js가 필요했죠.

오늘 배울 **GitHub Codespaces**는 이 문제를 한 번에 해결합니다!

## 핵심 개념: GitHub Codespaces란?

GitHub 저장소를 **브라우저에서 바로 열어서** 코딩하고 실행하는 **클라우드 환경**입니다.

```
  ┌────────────────────────────────────────────────────────┐
  │                    GitHub.com                           │
  │                                                        │
  │   ┌─────────────────┐     ┌──────────────────────────┐ │
  │   │  edu-padlet      │     │  Codespace (클라우드 PC)  │ │
  │   │  저장소           │ ──▶ │                          │ │
  │   │  (코드 보관함)    │     │  ✅ VS Code 설치됨       │ │
  │   └─────────────────┘     │  ✅ Node.js 설치됨        │ │
  │                           │  ✅ 터미널 사용 가능       │ │
  │          버튼 하나로       │  ✅ 서버 실행 가능         │ │
  │          바로 시작!        │  ✅ 공개 URL 생성 가능     │ │
  │                           └──────────────────────────┘ │
  └────────────────────────────────────────────────────────┘
```

### 비유로 이해하기: 클라우드 PC방

PC방에 가면 게임이 이미 설치되어 있죠?
내 컴퓨터에는 게임이 없어도, PC방에 가면 바로 할 수 있습니다.

**Codespaces는 코딩에 필요한 모든 것이 이미 설치된 클라우드 PC방이에요.**

- 내 컴퓨터에 Node.js 없어도 OK
- 내 컴퓨터에 VS Code 없어도 OK
- 브라우저만 열면 바로 코딩 시작!
- 심지어 태블릿이나 크롬북에서도 됩니다

## Codespaces 화면 구성

Codespaces를 열면 **VS Code와 똑같은 화면**이 브라우저에 나타납니다.

```
  ┌──────────────────────────────────────────────────────────┐
  │  브라우저 탭: github.dev                                   │
  ├──────────┬───────────────────────────────────────────────┤
  │          │                                               │
  │  📁 파일  │           편집기 (코드 작성)                    │
  │  목록     │                                               │
  │          │   server.js                                   │
  │  public/ │   ─────────────────────────                   │
  │  server  │   const express = require(...)                │
  │  db.js   │   ...                                         │
  │  .env    │                                               │
  │          │                                               │
  │          ├───────────────────────────────────────────────┤
  │          │  터미널 (명령어 입력)                            │
  │          │  $ node server.js                              │
  │          │  ✅ 서버가 3000번 포트에서 시작됨                │
  │          ├───────────────────────────────────────────────┤
  │          │  PORTS │ 3000  │ Public │ 🌐 열기              │
  └──────────┴───────────────────────────────────────────────┘
```

Step 1~3에서 VS Code를 사용해봤다면, **이미 다 아는 화면**입니다!

## 왜 Codespaces인가? (다른 방법과 비교)

| 항목 | 내 컴퓨터에서 실행 | Codespaces |
|------|-----------------|------------|
| Node.js 설치 | 필요함 | 이미 설치됨 |
| VS Code | 설치 필요 | 브라우저에서 바로 |
| 서버 실행 | localhost (나만) | 공개 URL (누구나) |
| 친구에게 공유 | 불가능 | 링크 하나로 공유 |
| 설치 시간 | 10~20분 | 30초~1분 |
| 크롬북/태블릿 | 어려움 | 가능! |

### 실제 개발자도 사용합니다

Codespaces는 학습용 도구가 아닙니다.
실제 회사에서 개발자들이 매일 사용하는 도구입니다.
지금 배우는 것이 곧 실무 경험이에요.

## 무료 사용량 안내

GitHub Codespaces는 **무료**입니다.

- 매월 **60시간** 무료 (2코어 기준)
- 저장 공간 **15GB** 무료
- 수업 시간 기준 약 **30~40회 수업**분 (한 수업 1.5시간 기준)
- 매월 1일에 사용량이 리셋됩니다

> 💡 **중요:** 사용하지 않을 때는 반드시 Codespace를 **중지(Stop)**해야 합니다!
> 안 끄면 시간이 계속 소모됩니다.
