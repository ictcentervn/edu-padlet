# AI와 함께 코딩하기 — 바이브코딩 프롬프트 모음

> **대상:** 중학교 1학년 ~ 고등학교 2학년 (수준별 표시 포함)

---

## 바이브코딩이란?

**바이브코딩(Vibe Coding)** = AI에게 한국어로 원하는 것을 설명해서 코드를 만드는 방법

코딩 문법을 몰라도 괜찮습니다. 내가 원하는 것을 **정확하게 설명**하면 AI가 코드를 만들어 줍니다.

```
나: "카드를 추가하고 삭제할 수 있는 보드 앱 만들어줘"
AI: "네, 여기 코드입니다!" → 코드 생성
```

핵심은 **좋은 질문(프롬프트)을 하는 것**입니다.

---

## 추천 도구 소개

### Bolt.new — 앱 뼈대 만들기

- **주소:** https://bolt.new
- **특징:** 브라우저에서 바로 앱을 만들 수 있습니다. 설치 없이!
- **사용법:** Bolt.new 접속 → 프롬프트 입력 → 코드 자동 생성 → 미리보기 확인

### Claude.ai — 코드 이해 + 수정 + 설명

- **주소:** https://claude.ai
- **특징:** 코드를 붙여넣고 질문하면 설명해 줍니다. 수정도 해줍니다.
- **사용법:** Claude.ai 접속 → 코드 + 질문 입력 → 답변 확인

### 사용 순서

1. **Bolt.new**에서 기본 프롬프트로 앱 뼈대 만들기
2. 만들어진 코드를 **Claude.ai**에 붙여넣고 이해하기
3. 수정하고 싶은 부분을 **Claude.ai**에 요청하기

## 🟢 기본 프롬프트 (Bolt.new용) — 앱 전체 생성

> **🟢 중학생도 할 수 있어요**

Bolt.new에서 새 프로젝트를 열고, 아래 프롬프트를 **그대로 붙여넣으세요:**

```
교육용 보드 앱을 만들어주세요.

요구사항:
- Express.js 서버 (Node.js)
- 카드를 추가하고 삭제할 수 있는 보드 UI
- HTML/CSS/JavaScript로 만든 프론트엔드
- 카드에는 제목, 내용, 색상(노랑/파랑/초록/분홍) 선택 가능
- Notion API를 사용해서 데이터 저장
- 한국어 주석 포함

파일 구조:
- server.js (Express 서버)
- db.js (Notion 연동)
- public/index.html
- public/style.css
- public/app.js
- .env.example
- package.json

API 엔드포인트:
- GET /api/cards — 카드 목록 조회
- POST /api/cards — 카드 생성
- PATCH /api/cards/:id — 카드 수정
- DELETE /api/cards/:id — 카드 삭제

디자인:
- 밝고 깔끔한 카드 UI
- 카드를 격자(grid) 형태로 배치
- 반응형 디자인 (모바일에서도 보기 좋게)
```

> 💡 이 프롬프트를 넣으면 우리가 수업에서 만든 것과 비슷한 앱이 자동으로 생성됩니다!

## 🟡 수정 프롬프트 모음 (Claude.ai용)

> **🟡 고등학생 도전!**

이미 만들어진 코드가 있을 때, 기능을 추가하거나 수정하고 싶으면 Claude.ai에 아래처럼 요청하세요.

### 1. 카드 수정 기능 추가

```
아래 코드에 카드 수정 기능을 추가해주세요.

현재 동작: 카드를 추가하고 삭제만 할 수 있습니다.
원하는 동작: 카드를 더블클릭하면 수정 모드가 되고, 제목과 내용을 수정한 후 저장 버튼을 누르면 업데이트됩니다.

서버에 PATCH /api/cards/:id 엔드포인트를 추가해주세요.
수정한 부분에 한국어 주석을 달아주세요.

[여기에 기존 코드 붙여넣기]
```

### 2. 코드 이해 요청

```
아래 코드가 무슨 일을 하는지 중학생이 이해할 수 있게 설명해줘.
각 줄마다 한국어로 주석을 달아주고,
전체 흐름을 순서대로 정리해줘.

[여기에 코드 붙여넣기]
```

### 3. 버그 수정 요청

```
아래 코드를 실행하면 이런 에러가 나요:

[에러 메시지 붙여넣기]

왜 이 에러가 나는지 알려주고, 어떻게 고치면 되는지 알려줘.
수정된 코드도 보여줘.

[여기에 코드 붙여넣기]
```

### 4. 스타일 변경 요청

```
아래 CSS를 수정해서 카드 디자인을 바꿔주세요.

원하는 스타일:
- 카드에 그림자(shadow) 효과 추가
- 카드 위에 마우스를 올리면 살짝 커지는 애니메이션
- 색상별로 카드 테두리 색 변경
- 둥근 모서리 (border-radius: 12px)

[여기에 CSS 코드 붙여넣기]
```

## 🔴 심화 도전 프롬프트 — 직접 작성해보기

> **🔴 심화: 직접 수정해보기**

아래 기능 중 하나를 골라서, **스스로 프롬프트를 작성**해 보세요.

### 도전 과제

1. **카드 드래그 이동:** 마우스로 카드를 잡아서 위치를 바꾸는 기능
2. **보드 여러 개 만들기:** "과학", "수학", "영어" 등 보드를 추가하고 전환하는 기능
3. **카드 검색:** 제목이나 내용에 특정 단어가 포함된 카드만 보여주는 기능
4. **카드 정렬:** 날짜순, 색상별로 카드를 정렬하는 기능
5. **다크 모드:** 배경을 어둡게 바꾸는 토글 버튼

### 좋은 프롬프트 작성 팁

| 나쁜 프롬프트 | 좋은 프롬프트 |
|-------------|-------------|
| "검색 기능 만들어줘" | "카드 제목에 입력한 단어가 포함된 카드만 필터링해서 보여주는 검색창을 추가해줘. 검색창은 보드 상단에 위치하고, 입력할 때마다 실시간으로 필터링돼야 해." |
| "디자인 바꿔줘" | "카드 배경색을 파스텔 톤으로 바꾸고, 카드 사이 간격을 16px로 맞추고, 전체 배경은 연한 회색(#f5f5f5)으로 해줘." |
| "에러 고쳐줘" | "node server.js를 실행하면 'Cannot find module express' 에러가 나요. 에러 전체 메시지는 이거예요: [메시지]. npm install은 했는데 왜 안 되는지 알려줘." |

**핵심:** 원하는 것을 **구체적으로**, **어디에** 나타나야 하는지, **어떻게** 동작해야 하는지 적어주세요.
