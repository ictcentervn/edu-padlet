# Notion API 키 발급하기 — 내 앱이 Notion에 접근하는 허가증

> **대상:** 중학교 1학년 ~ 고등학교 2학년

---

## API 키가 왜 필요한가요?

식당 회원카드를 생각해 보세요.

- 회원카드가 없으면 → 주문을 할 수 없습니다.
- 회원카드가 있으면 → 식당에서 내가 누구인지 알고, 주문을 처리해 줍니다.

**API 키**는 우리 앱의 회원카드입니다.

```
우리 앱: "Notion, 카드 데이터 줘!"
Notion:  "넌 누구야? 회원카드(API 키) 보여줘."
우리 앱: "여기! secret_abc123..."
Notion:  "확인! 여기 데이터야."
```

API 키가 없으면 Notion은 우리 앱의 요청을 거부합니다.

---

## 1단계: Notion 계정 만들기 / 로그인

**아직 계정이 없다면:**

1. 브라우저에서 **notion.so** 접속
2. **"Get Notion free"** 클릭
3. 이메일 입력 → 무료 플랜 선택
4. 이메일로 온 인증 메일 확인 → 인증 완료

**이미 계정이 있다면:** 바로 2단계로 넘어가세요!

> 💡 **교사 메모:** 학교 이메일이 없는 학생은 개인 이메일(Gmail 등)을 사용해도 됩니다. Notion 무료 플랜으로 충분합니다.

---

## 2단계: Integration(연동) 만들기

> **Integration이란?** 외부 프로그램(우리가 만드는 앱)이 Notion에 접근할 수 있도록 허가증을 발급하는 것입니다.

1. Notion에 로그인한 상태로 아래 주소에 접속합니다:

   **https://www.notion.so/my-integrations**

2. **"+ New integration"** 버튼을 클릭합니다

3. 아래처럼 입력합니다:
   - **Name:** `우리반 패들렛` (다른 이름도 괜찮아요)
   - **Associated workspace:** 본인 워크스페이스 선택
   - **Type:** Internal (기본값 그대로 두세요)

4. **"Submit"** 버튼을 클릭합니다

> ✅ "우리반 패들렛" Integration이 만들어졌습니다!

---

## 3단계: API 키 복사하기

1. Integration 설정 페이지에서 **"Internal Integration Secret"** 항목을 찾습니다
2. **눈 모양 아이콘**을 클릭하면 키가 보입니다
3. `secret_`으로 시작하는 긴 문자열이 보이면 성공!
4. **"복사"** 버튼을 클릭합니다

```
예시: secret_AbCdEfGhIjKlMnOpQrStUvWxYz1234567890
```

5. 메모장이나 임시 파일에 붙여넣어 두세요 (나중에 .env 파일에 넣을 겁니다)

> ⚠️ **중요!** 이 키는 **절대 다른 사람에게 공유하지 마세요.** 비밀번호와 같습니다. 이 키를 가진 사람은 여러분의 Notion 데이터에 접근할 수 있습니다.

---

## 4단계: Notion 데이터베이스 만들기

1. Notion 왼쪽 사이드바에서 **"+ Add a page"** 클릭
2. 페이지 제목을 **`패들렛 DB`**라고 입력합니다
3. 본문을 클릭하고 `/database`를 입력한 뒤 → **"Table view"** 선택

4. 아래 컬럼(열)을 추가합니다:

   | 순서 | 컬럼 이름 | 타입(Type) | 설명 |
   |------|---------|-----------|------|
   | 기본 | Title | Title (기본 제공) | 카드 제목 |
   | 추가 1 | Content | Text | 카드 내용 |
   | 추가 2 | Color | Select | 카드 색상 |
   | 추가 3 | BoardId | Text | 보드 구분용 |

5. **Color** 컬럼의 옵션(Options)에 아래 4가지를 추가합니다:
   - `yellow`
   - `blue`
   - `green`
   - `pink`

> **컬럼 추가 방법:** 테이블 오른쪽 끝의 **"+"** 버튼 클릭 → 컬럼 이름 입력 → 타입 선택

> ✅ 4개의 컬럼이 있는 테이블이 완성되었으면 성공!

---

## 5단계: Integration과 DB 연결하기

Integration을 만들었어도, 어떤 DB에 접근할지 **직접 허용**해줘야 합니다.

1. 방금 만든 **`패들렛 DB`** 페이지를 엽니다
2. 우측 상단 **"..."** (점 세 개, 더보기) 버튼을 클릭합니다
3. 메뉴에서 **"Connections"** 를 클릭합니다
4. **"우리반 패들렛"** (2단계에서 만든 Integration 이름)을 찾아 선택합니다
5. **"Confirm"** 을 클릭합니다

```
[패들렛 DB 페이지]  ←──연결──→  [우리반 패들렛 Integration]
```

> ✅ "우리반 패들렛" Integration이 이 DB에 접근할 수 있게 되었습니다!

> ⚠️ **이 단계를 빠뜨리면** 나중에 `object_not_found` 에러가 납니다. 꼭 확인하세요!

---

## 6단계: 데이터베이스 ID 찾기

1. **`패들렛 DB`** 페이지가 열린 상태에서 **브라우저 주소창**을 봅니다

2. URL이 이런 형태입니다:
   ```
   https://www.notion.so/내워크스페이스/abc123def456abc123def456abc12345?v=xyz789...
   ```

3. `?v=` **앞에 있는 32자리 문자열**이 데이터베이스 ID입니다
   - 예: `abc123def456abc123def456abc12345`
   - `?v=` 뒤의 내용은 복사하지 마세요!

4. 이 ID도 메모장에 붙여넣어 두세요

> ✅ API 키와 DB ID, 두 가지를 모두 확보했으면 성공!

## .env 파일에 넣는 방법

프로젝트 폴더에서 `.env.example` 파일을 `.env`로 복사합니다:

```bash
cp .env.example .env
```

그리고 `.env` 파일을 열어서 실제 값을 넣습니다:

```
NOTION_API_KEY=secret_여기에_3단계에서_복사한_키_붙여넣기
NOTION_DATABASE_ID=여기에_6단계에서_복사한_ID_붙여넣기
```

**완성된 .env 파일 예시:**
```
NOTION_API_KEY=secret_AbCdEfGhIjKlMnOpQrStUvWxYz1234567890
NOTION_DATABASE_ID=abc123def456abc123def456abc12345
```

> ⚠️ **주의:** 등호(`=`) 앞뒤에 공백을 넣지 마세요. 따옴표도 넣지 마세요.

---

## ⚠️ 주의사항

1. **API 키를 친구에게 공유하지 마세요.** 비밀번호와 같습니다.
2. **API 키를 GitHub 등에 올리지 마세요.** `.env` 파일은 `.gitignore`에 추가해야 합니다.
3. **각 학생이 자기 Integration을 직접 만들어야 합니다.** 한 명이 만들어서 공유하면 안 됩니다.

---

> 💡 **교사 메모:**
> - 수업 전에 선생님이 1~6단계를 미리 한 번 해보시길 권장합니다.
> - Notion 무료 계정의 API 호출 횟수는 넉넉하므로 걱정하지 않으셔도 됩니다.
> - 학생들이 가장 자주 빠뜨리는 단계는 **5단계(Integration-DB 연결)**입니다. 수업 중 반드시 확인해 주세요.
> - 학생 계정 없이 교사 계정 하나로 시연만 하려면, 교사 Integration 하나로 진행할 수 있습니다.
