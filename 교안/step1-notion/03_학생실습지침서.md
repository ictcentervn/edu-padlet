# 따라하면 완성! Step 1 실습 가이드

> **대상:** 중학교 1학년 ~ 고등학교 2학년
> 이 문서를 **위에서 아래로 순서대로** 따라하면 앱이 완성됩니다.

---

## 시작 전 체크리스트

아래 항목을 모두 확인하고 시작하세요:

- [ ] 노트북 충전 상태 확인 (50% 이상)
- [ ] Wi-Fi 연결 확인
- [ ] **Node.js** 설치 완료
- [ ] **VS Code** 설치 완료
- [ ] **Notion 계정** 로그인 완료
- [ ] Chrome 또는 Edge 브라우저 준비

### Node.js 설치 확인 방법

터미널(또는 명령 프롬프트)을 열고 아래를 입력하세요:

```bash
node --version
```

`v18.0.0` 이상의 숫자가 나오면 OK!

```
예시 결과: v20.11.0   ← 이렇게 나오면 성공!
```

만약 `node: command not found` 또는 아무 반응이 없으면:
→ Node.js가 설치되지 않았습니다.
→ https://nodejs.org 에서 **LTS 버전**을 다운로드해서 설치하세요.

> ✅ **여기까지 확인 포인트:** `node --version`이 정상적으로 나오면 다음으로!

## A단계: 프로젝트 준비

### A-1. 프로젝트 폴더 받기

선생님이 나눠준 프로젝트 파일을 바탕화면 또는 원하는 위치에 저장합니다.

폴더 이름: `step1-notion`

### A-2. VS Code에서 열기

1. VS Code를 실행합니다
2. 상단 메뉴에서 **File** → **Open Folder** 클릭
3. `step1-notion` 폴더를 선택하고 **열기** 클릭

### A-3. 터미널 열기

VS Code 상단 메뉴에서:

**Terminal** → **New Terminal**

화면 아래쪽에 검은색(또는 어두운) 창이 열립니다. 여기에 명령어를 입력합니다.

> ✅ **여기까지 확인 포인트:** VS Code 왼쪽에 파일 목록이 보이고, 아래에 터미널이 열려 있으면 성공!

---

## B단계: 코드 파일 준비

### B-1. 파일 구조 확인

VS Code 왼쪽 탐색기에서 아래 파일들이 보이는지 확인하세요:

```
step1-notion/
  ├── server.js          ← 서버 코드 (주방)
  ├── db.js              ← DB 연동 코드 (Notion과 대화)
  ├── package.json       ← 필요한 도구 목록
  ├── .env.example       ← 환경변수 예시 파일
  └── public/            ← 브라우저에서 보이는 파일들
       ├── index.html
       ├── style.css
       └── app.js
```

### B-2. npm install 실행

터미널에 아래 명령어를 입력하고 Enter를 누릅니다:

```bash
npm install
```

잠시 기다리면 설치가 완료됩니다. `node_modules` 폴더가 새로 생기면 성공!

```
예시 출력:
added 57 packages in 3s
```

> ⚠️ 에러가 나면? → `step1-notion` 폴더 안에서 실행하고 있는지 확인하세요. 터미널에 `ls`를 입력했을 때 `package.json`이 보여야 합니다.

> ✅ **여기까지 확인 포인트:** `node_modules` 폴더가 생겼으면 성공!

## C단계: Notion 설정

Notion API 키 발급과 DB 만들기가 필요합니다.

**[01_API키설정가이드.md](./01_API키설정가이드.md)** 를 열고 1단계~6단계를 모두 따라하세요.

완료되면 아래 두 가지 값을 메모장에 가지고 있어야 합니다:
- **NOTION_API_KEY:** `secret_` 으로 시작하는 문자열
- **NOTION_DATABASE_ID:** 32자리 문자열

> ✅ **여기까지 확인 포인트:** API 키와 DB ID 두 가지를 메모장에 복사해 두었으면 성공!

## D단계: .env 파일 만들기

### D-1. .env.example 복사

터미널에 아래 명령어를 입력합니다:

**Mac:**
```bash
cp .env.example .env
```

**Windows:**
```bash
copy .env.example .env
```

### D-2. .env 파일 열기

VS Code 왼쪽 탐색기에서 `.env` 파일을 클릭해서 엽니다.

> 파일이 안 보이면? VS Code 왼쪽 아래 톱니바퀴 → Settings → "files.exclude" 검색 → `**/.env` 항목 제거

### D-3. 실제 값 입력

`.env` 파일을 아래처럼 수정합니다:

```
NOTION_API_KEY=secret_여기에_실제_키_붙여넣기
NOTION_DATABASE_ID=여기에_실제_ID_붙여넣기
```

**주의사항:**
- `=` 앞뒤에 **공백 없이** 입력하세요
- 따옴표(`"`, `'`)를 넣지 마세요
- 줄 끝에 공백이 들어가지 않게 주의하세요

**올바른 예:**
```
NOTION_API_KEY=secret_AbCdEfGhIjKlMnOpQrStUvWxYz1234567890
NOTION_DATABASE_ID=abc123def456abc123def456abc12345
```

**잘못된 예:**
```
NOTION_API_KEY = secret_abc123     ← 공백 있으면 안 됨!
NOTION_API_KEY="secret_abc123"     ← 따옴표 있으면 안 됨!
```

파일을 저장합니다 (Ctrl+S 또는 Cmd+S).

> ✅ **여기까지 확인 포인트:** `.env` 파일에 두 줄이 채워져 있으면 성공!

## E단계: 서버 실행 및 확인

### E-1. 서버 실행

터미널에 아래 명령어를 입력합니다:

```bash
node server.js
```

성공하면 아래 메시지가 나옵니다:

```
✅ Notion DB 연결 성공!

✅ 서버가 시작되었습니다!
   브라우저에서 http://localhost:3000 을 열어보세요
```

> ⚠️ 에러가 나면? → 05_트러블슈팅.md를 확인하세요!

### E-2. 브라우저에서 접속

1. Chrome 또는 Edge 브라우저를 엽니다
2. 주소창에 아래를 입력합니다:

```
http://localhost:3000
```

3. 우리 반 패들렛 화면이 나타납니다!

### E-3. 카드 추가해보기

1. **"새 카드 만들기"** 버튼을 클릭합니다
2. 제목과 내용을 입력합니다
3. 색상을 선택합니다
4. **저장**을 클릭합니다

화면에 카드가 나타나면 성공!

### E-4. Notion에서 데이터 확인

1. 브라우저에서 **Notion**을 엽니다
2. **패들렛 DB** 페이지로 이동합니다
3. 방금 추가한 카드의 데이터가 테이블에 보이는지 확인합니다

```
Title        │ Content    │ Color  │ BoardId
─────────────┼────────────┼────────┼─────────
오늘 할일     │ 수학숙제    │ yellow │ default
```

> ✅ **여기까지 확인 포인트:** 브라우저에서 카드가 보이고, Notion DB에도 데이터가 들어가 있으면 **완성**!
