## AI가 만든 코드 이해하기

AI가 코드를 만들어줬다면, 바로 붙여넣기 전에 이해하고 싶다면:

```
이 코드에서 [기능 이름]이 어떻게 동작하는지 중학생이 이해할 수 있게 설명해줘:
[코드 붙여넣기]
```

**Glitch에서의 작업 흐름:**

```
① Claude.ai에 프롬프트 입력
     │
     ▼
② AI가 코드를 생성
     │
     ▼
③ Glitch에서 해당 파일을 열고 코드 붙여넣기
     │
     ▼
④ 자동 저장 → 자동 재시작 → Preview에서 즉시 확인
     │
     ├── 동작하면 → 다음 기능 도전!
     └── 에러나면 → Logs 탭에서 에러 확인 → AI에게 에러 메시지 보여주고 수정 요청
```

## 프롬프트 작성 팁

1. **현재 코드를 함께 보여주세요.** AI가 기존 코드에 맞는 답을 줍니다.
2. **한 번에 하나의 기능만 요청하세요.** 여러 기능을 한꺼번에 요청하면 복잡해집니다.
3. **"한국어 주석 포함"을 꼭 넣으세요.** 코드를 이해하는 데 도움이 됩니다.
4. **에러가 나면 에러 메시지를 그대로 AI에게 보여주세요.** AI가 고쳐줍니다.
