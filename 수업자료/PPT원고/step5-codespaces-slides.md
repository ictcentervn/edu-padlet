# Step 5: GitHub Codespaces — 브라우저에서 코딩하기 — PPT 텍스트 원고

> 총 12장 | 선생님이 Canva/Google Slides/PowerPoint에서 슬라이드를 만들 때 참고할 원고입니다.

---
## 슬라이드 1: 설치 없이 코딩하려면?

**핵심 텍스트:**
- 학교 컴퓨터에 Node.js 설치 금지?
- 집에 컴퓨터 없이 태블릿만?
- localhost:3000은 내 컴퓨터에서만 보인다
- GitHub Codespaces가 전부 해결!

**시각 자료 제안:** 3가지 문제 상황 아이콘 (설치 금지 표시판 / 태블릿 / "나만 보임" 자물쇠) + 가운데 Codespaces 로고 + "전부 해결!" 화살표

**선생님 멘트:** "학교 컴퓨터에 프로그램 설치가 안 된다면 어떡하죠? 친구에게 내 앱을 보여주고 싶은데 localhost는 내 컴퓨터에서만 되죠. 오늘 배울 Codespaces는 이 문제를 모두 해결합니다."

---
## 슬라이드 2: Codespaces = 클라우드 PC방

**핵심 텍스트:**
- PC방에 가면 게임이 이미 설치되어 있다
- Codespaces = 코딩에 필요한 것이 모두 설치된 클라우드 PC방
- Node.js 설치 불필요, VS Code 설치 불필요
- 브라우저만 열면 바로 코딩 시작!

**시각 자료 제안:** 왼쪽: PC방 간판 + 모니터에 게임 화면. 오른쪽: 브라우저 안에 VS Code 화면. "이미 다 설치되어 있어요!" 말풍선

**선생님 멘트:** "PC방에 가면 게임이 이미 설치되어 있어서 바로 할 수 있죠? Codespaces도 같습니다. 브라우저를 열면 VS Code도, Node.js도, 터미널도 전부 준비되어 있어요. 클라우드 PC방인 거예요."

---
## 슬라이드 3: 주의! github.dev와 Codespace는 다릅니다

**핵심 텍스트:**
- github.dev (마침표 키) = 코드 편집만 가능
- Codespace = 코드 편집 + 서버 실행 + 터미널 사용
- 우리는 서버를 실행해야 하니까 Codespace를 사용!
- 헷갈리지 마세요!

**시각 자료 제안:** 좌우 비교표. github.dev: "편집만" (가위 아이콘). Codespace: "편집 + 실행 + 터미널" (로켓 아이콘). Codespace 쪽에 "이걸 쓸 거예요!" 표시

**선생님 멘트:** "주의할 점이 하나 있어요. GitHub 저장소에서 마침표 키를 누르면 github.dev가 열리는데, 이건 코드 편집만 됩니다. 서버를 실행하려면 Codespace를 따로 만들어야 해요. 헷갈리지 마세요!"

---
## 슬라이드 4: Codespace 만들기

**핵심 텍스트:**
- GitHub 저장소 페이지에서 Code 버튼 클릭
- Codespaces 탭 선택
- "Create codespace on main" 클릭
- 30초~1분 후 VS Code 화면이 브라우저에 나타남!

**시각 자료 제안:** GitHub 저장소 페이지 스크린샷 위치 표시. Code 버튼 -> Codespaces 탭 -> Create 버튼 순서를 번호와 화살표로 표시

**선생님 멘트:** "지금부터 Codespace를 만들어 봅시다. GitHub 저장소 페이지에서 Code 버튼을 누르고, Codespaces 탭에서 만들기 버튼을 누르세요. 잠시 기다리면 브라우저에 VS Code가 나타납니다!"

---
## 슬라이드 5: 실습 — 서버 실행 + 포트 공개

**핵심 텍스트:**
- 터미널에서 npm install 실행
- node server.js로 서버 시작
- 포트 알림이 뜨면 "Open in Browser" 클릭
- PORTS 탭에서 포트를 "Public"으로 변경

**시각 자료 제안:** Codespace 화면 스크린샷 위치 표시. 터미널 영역 + 포트 알림 팝업 + PORTS 탭. 번호 순서대로 화살표

**선생님 멘트:** "터미널에서 서버를 실행하면 포트 알림이 뜹니다. 'Open in Browser'를 클릭하면 앱이 열려요. 그리고 중요! PORTS 탭에서 포트를 Public으로 바꿔야 친구도 접속할 수 있습니다."

---
## 슬라이드 6: URL로 친구에게 공유!

**핵심 텍스트:**
- Codespace가 생성한 공개 URL 복사
- 형태: https://xxx-3000.app.github.dev
- 이 URL을 친구에게 카톡으로 보내기
- localhost가 아니라 인터넷 주소!

**시각 자료 제안:** 지금까지 vs 이제부터 비교. localhost:3000(나만 보임, X) vs xxx-3000.app.github.dev(누구나 보임, 체크). 카톡 말풍선에 URL을 보내는 모습

**선생님 멘트:** "이 URL을 복사해서 옆 친구에게 카톡으로 보내 보세요. 친구가 클릭하면 여러분의 앱이 보일 거예요! localhost는 나만 보이지만, 이 URL은 누구나 접속할 수 있습니다."

---
## 슬라이드 7: GitHub 인증 "Continue" 안내

**핵심 텍스트:**
- 친구가 URL을 열면 "Continue" 버튼이 나올 수 있음
- GitHub 로그인 상태면 자동으로 넘어감
- 비로그인 시에도 Continue 버튼 클릭하면 OK
- 당황하지 마세요!

**시각 자료 제안:** "Continue to..." 화면 스크린샷 위치 표시. "이 화면이 나오면 Continue 클릭!" 안내 화살표

**선생님 멘트:** "친구가 URL을 열었는데 이런 화면이 나올 수 있어요. 당황하지 말고 Continue 버튼을 누르면 됩니다. GitHub이 '이 사이트에 접속하시겠어요?'라고 확인하는 거예요."

---
## 슬라이드 8: Live Share로 실시간 협업

**핵심 텍스트:**
- Live Share = 같은 코드를 동시에 편집
- 비유: 구글 Docs에서 함께 문서 작성하는 것
- 친구의 커서가 실시간으로 보임
- Codespace에서 Live Share 확장 프로그램 사용

**시각 자료 제안:** 코드 편집기에 두 개의 커서(다른 색상)가 있는 모습. "김민수 커서", "이서연 커서" 라벨. 구글 Docs 협업 화면과 나란히 배치

**선생님 멘트:** "구글 Docs에서 친구와 동시에 문서를 편집해 본 적 있죠? Live Share를 쓰면 코드도 같이 편집할 수 있어요. 친구의 커서가 실시간으로 보입니다. 짝 프로그래밍을 해 보세요!"

---
## 슬라이드 9: 무료 사용량 안내

**핵심 텍스트:**
- 매월 60시간 무료 (2코어 기준)
- 수업 약 30~40회분 (1.5시간/회 기준)
- 매월 1일에 리셋
- 넉넉하지만, 안 쓸 때는 꼭 중지!

**시각 자료 제안:** 배터리/시간 게이지 바. "60시간/월" 표시. "수업 30~40회분" 라벨

**선생님 멘트:** "무료로 매월 60시간 쓸 수 있어요. 수업 시간으로 치면 30~40회분이니 넉넉합니다. 하지만 안 쓸 때는 반드시 중지해야 시간이 낭비되지 않아요."

## 슬라이드 10: Codespace 중지 필수!

**핵심 텍스트:**
- 수업 끝나면 반드시 Codespace 중지(Stop)!
- github.com/codespaces에서 확인
- 점 3개 메뉴 -> Stop codespace
- 안 끄면 무료 시간이 계속 소모됨!

**시각 자료 제안:** Codespace 관리 페이지 스크린샷 위치 표시. "Stop codespace" 메뉴 강조. 경고 아이콘 + "안 끄면 시간 소모!" 빨간 텍스트

**선생님 멘트:** "이건 꼭 기억하세요! 수업이 끝나면 반드시 Codespace를 중지해야 합니다. 안 끄면 시간이 계속 흘러갑니다. 지금 바로 중지하는 방법을 알려드릴게요."

## 슬라이드 11: 오늘 배운 것 정리

**핵심 텍스트:**
- Codespaces = 클라우드 PC방 (브라우저에서 코딩+실행)
- 설치 없이 서버 실행 가능
- 공개 URL로 친구에게 앱 공유
- 수업 후 Codespace 중지 필수!

**시각 자료 제안:** 4개 키워드 카드. 클라우드 PC방/설치 불필요/URL 공유/중지 필수. 각각 아이콘

**선생님 멘트:** "오늘 핵심을 정리합니다. Codespaces는 브라우저에서 코딩하고 실행하는 클라우드 PC방입니다. 설치가 필요 없고, URL로 공유할 수 있고, 수업 후에는 반드시 중지하세요!"
