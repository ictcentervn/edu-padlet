# 교육용 Padlet 클론 6단계 프로젝트 — 교사 지침서 전체 개요

## 프로젝트 한줄 요약
같은 웹 앱(패들렛 클론)을 6단계에 걸쳐 만들면서, 데이터베이스 > 코드 관리 > 클라우드 배포까지 풀스택 개발의 전체 흐름을 체험한다.

## 6단계 로드맵

| Step | 주제 | 핵심 개념 | 차시 | 난이도 |
|------|------|---------|------|--------|
| 1 | Notion API | DB, 서버, API, .env | 3차시 | 입문 |
| 2 | SQLite | SQL, 로컬 파일 DB, 모듈화 | 2차시 | 기본 |
| 3 | Supabase | 클라우드 DB, 다중 접속 | 2차시 | 중급 |
| 4 | GitHub | Git, 버전 관리, .gitignore | 2차시 | 기본 |
| 5 | Codespaces | 클라우드 개발 환경, 포트 공개 | 2차시 | 기본 |
| 6 | Render | 배포, CI/CD, 환경변수 | 2차시 | 심화 |
| | | **총합** | **13차시** | |

## 학년별 추천 경로

| 대상 | 추천 경로 | 비고 |
|------|---------|------|
| 중학교 1~3학년 | Step 1 + Step 4 + Step 5 | 핵심 체험 위주 (7차시) |
| 고등학교 1~2학년 | Step 1 + Step 2 + Step 4 + Step 5 | DB 비교 추가 (9차시) |
| 고등학교 2~3학년 (심화) | Step 1~6 전체 | 풀코스 (13차시) |
| 동아리/방과후 (단기) | Step 1 + Step 5 | 최소 경험 (5차시) |

## 필요한 계정 목록

| 계정 | 필요 Step | 가입 시 주의사항 |
|------|----------|----------------|
| Notion | Step 1 | 무료 플랜 OK, 이메일 인증 |
| GitHub | Step 3~6 | 이메일 인증 필수, 수업 전 가입 권장 |
| Supabase | Step 3, 6 | GitHub 계정으로 로그인, 무료 프로젝트 2개 제한 |
| Render | Step 6 | GitHub 계정으로 로그인 |

## 수업 시작 전 교사가 미리 해볼 것

- [ ] **Step 1**: Notion Integration 생성 > DB 만들기 > API 키 발급 > 서버 실행 > 카드 CRUD 확인
- [ ] **Step 2**: `npm install better-sqlite3` 빌드 성공 확인 (Windows 특히 주의)
- [ ] **Step 3**: Supabase 프로젝트 생성 > SQL 실행 > 서버 실행 > Table Editor 확인
- [ ] **Step 4**: Git 설치 > GitHub 저장소 생성 > push > .gitignore 확인
- [ ] **Step 5**: Codespace 생성 > 서버 실행 > 포트 Public > URL 접속 > Stop
- [ ] **Step 6**: Render 배포 > 환경변수 설정 > 배포 URL 접속 > 자동 배포 확인
- [ ] 학교 네트워크에서 github.com, supabase.com, render.com 접속 가능 여부 확인
- [ ] 각 Step 폴더의 프로젝트 파일이 완비되어 있는지 확인

## Step별 가장 흔한 에러 요약

| Step | 에러 Top 1 | 한 줄 해결법 |
|------|-----------|------------|
| 1 | Integration-DB 연결 누락 | DB 페이지 "..." > Connections 확인 |
| 2 | better-sqlite3 빌드 에러 | C++ 빌드 도구 설치 (xcode-select / windows-build-tools) |
| 3 | .env URL/Key 복사 오류 | Settings > Data API에서 복사 아이콘으로 복사 |
| 4 | .env 실수로 push | .gitignore 확인 + git rm --cached .env + 키 재발급 |
| 5 | .github.dev로 열림 (Codespace 아님) | github.com/codespaces에서 직접 생성 |
| 6 | 환경변수 누락 | Render 대시보드 Environment 탭에서 설정 |
