## 실습 체크리스트

### A단계: Codespace 만들기

- [ ] GitHub에서 내 `edu-padlet` 저장소 열기
- [ ] 초록색 "Code" 버튼 클릭
- [ ] "Codespaces" 탭 선택
- [ ] "Create codespace on main" 클릭
- [ ] 브라우저에 VS Code 화면이 나타남

### B단계: 서버 실행

- [ ] 터미널에서 `npm install` 실행
- [ ] `cp .env.example .env` 실행
- [ ] `.env` 파일에 Supabase URL + Key 입력
- [ ] `node server.js` 실행
- [ ] 앱이 브라우저 새 탭에서 열림

  앱이 열린 방법: [ ] 팝업 클릭 [ ] PORTS 탭에서 직접 열기

### C단계: 앱 동작 확인

- [ ] 카드 추가 성공
- [ ] 카드 삭제 성공
- [ ] 새로고침 후에도 데이터 유지 확인

### D단계: 포트 공개 + URL 공유

- [ ] PORTS 탭에서 3000번 포트 우클릭
- [ ] Port Visibility를 "Public"으로 변경
- [ ] URL 복사
