## 실습 체크리스트

단계별로 완료하면 체크하세요!

### A단계: 프로젝트 준비

- [ ] VS Code에서 `step1-notion` 폴더 열기
- [ ] 터미널(Terminal) 열기
- [ ] `node --version` 입력하여 Node.js 확인

  버전 결과: v____.__.__

### B단계: 패키지 설치

- [ ] 터미널에서 `npm install` 실행
- [ ] `node_modules` 폴더가 생겼는지 확인

### C단계: Notion 설정

- [ ] Notion 계정 로그인
- [ ] Integration(통합) 만들기
- [ ] 데이터베이스 만들기
- [ ] Integration과 데이터베이스 연결하기

### D단계: .env 파일 만들기

- [ ] `.env.example` 파일을 `.env`로 복사
- [ ] API 키와 DB ID를 `.env` 파일에 입력
- [ ] 저장 (Ctrl+S)

### E단계: 서버 실행 및 확인

- [ ] `node server.js` 실행
- [ ] 브라우저에서 `http://localhost:3000` 접속
- [ ] 카드 추가하기 (제목, 내용, 색상)
- [ ] Notion DB에서 내가 추가한 카드 데이터 확인
