## 실습 체크리스트

### A단계: Render 가입

- [ ] render.com 접속
- [ ] GitHub 계정으로 로그인
- [ ] Render 대시보드가 보이나요? [ ] 예 [ ] 아니오

### B단계: Web Service 생성

- [ ] "New +" --> "Web Service" 선택
- [ ] GitHub에서 `edu-padlet` 저장소 연결 (Connect)
- [ ] 서비스 설정 입력:
  - Name: ________________________
  - Region: Singapore
  - Branch: main
  - Build Command: `npm install`
  - Start Command: `node server.js`
  - Instance Type: Free

### C단계: 환경변수 설정

- [ ] SUPABASE_URL 입력 완료
- [ ] SUPABASE_ANON_KEY 입력 완료

### D단계: 배포 완료 확인

- [ ] "Create Web Service" 클릭
- [ ] 빌드 로그에서 "Your service is live!" 확인 (2~3분 대기)
- [ ] 내 URL에서 앱이 보이나요? [ ] 예 [ ] 아니오
- [ ] 카드를 추가해보세요 -- 성공했나요? [ ] 예 [ ] 아니오

  바꾼 내용: ________________________________________________________________

### GitHub에 올리기

- [ ] `git add .` 실행
- [ ] `git commit -m "________________________"` 실행
- [ ] `git push origin main` 실행

### Render 자동 배포 확인

- [ ] Render 대시보드 --> "Deploys" 탭에서 새 배포 시작 확인
