## 실습 체크리스트

### A단계: GitHub 가입 + Git 설정

- [ ] GitHub 가입 완료 (또는 이미 가입됨)
- [ ] `git --version` 확인 -- 버전: ________________
- [ ] `git config --global user.name "내 이름"` 설정
- [ ] `git config --global user.email "내 이메일"` 설정

### B단계: git init + .gitignore

- [ ] 프로젝트 폴더에서 `git init` 실행
- [ ] `.gitignore` 파일 만들기

### C단계: GitHub 저장소 만들기

- [ ] GitHub에서 `edu-padlet` 저장소 만들기 (Public)

### D단계: add, commit, push

- [ ] `git add .` 실행
- [ ] `git commit -m "첫 번째 커밋: 우리 반 패들렛"` 실행
- [ ] `git remote add origin https://github.com/______/edu-padlet.git`
- [ ] `git branch -M main` 실행
- [ ] `git push -u origin main` 실행

### E단계: GitHub에서 확인

- [ ] GitHub에서 내 코드가 보이나요? [ ] 예 [ ] 아니오
- [ ] `.env` 파일이 보이지 않나요? [ ] 맞음 (안 보임) [ ] 틀림 (보임!)
