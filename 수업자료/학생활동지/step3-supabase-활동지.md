## 실습 체크리스트

### A단계: Supabase 설정

- [ ] https://supabase.com 접속
- [ ] GitHub 계정으로 로그인
- [ ] 새 프로젝트 만들기 (이름: `edu-padlet`)
- [ ] SQL Editor에서 테이블 만들기 (CREATE TABLE 실행)
- [ ] Table Editor에서 cards, boards 테이블 확인
- [ ] Project URL 복사
- [ ] anon public key 복사

### B단계: 프로젝트 준비

- [ ] `step3-supabase` 폴더 만들기
- [ ] Step 2에서 `server.js`와 `public/` 폴더 복사
- [ ] `npm init -y` 실행
- [ ] `npm install express cors dotenv @supabase/supabase-js` 실행
- [ ] `.env` 파일 만들고 URL + Key 입력
- [ ] `db.js` 파일 새로 만들기 (코드 붙여넣기)

### C단계: 서버 실행
